About this Project

This project is about the data virtualization of UBC Happiness.

This demo is a illustration of psychological health data collected from UBC Vancouver and Okanagan Campus.

As a preliminary research, the data process is conducted by front-end JavaScript, which is actually running on your own browser. To this end, the speed of processing is a bit slow after you select the data set.
